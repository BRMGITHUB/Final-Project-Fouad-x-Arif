

## Start the server


```bash
npm run dev
```

## Building

To Build

```bash
npm run build
```

## Create an env File

To run the Supabase client, create a .env file in the root folder and add 

```bash
VITE_PUBLIC_SUPABASE_URL={supabaseApiUrl}
VITE_PUBLIC_SUPABASE_ANON_KEY={supabaseApiPublicAnonKey}
```
